### Đối tượng sử dụng
- Quản trị viên
- Khách hàng

### Chức năng từng đối tượng
A. Quản trị viên
- Quản lý sản phẩm
- Quản lý đơn hàng
- Quản lý danh mục sản phẩm
- Quản lý các đơn hàng
- Quản lý tài khoản khách hàng
- Quản lý báo cáo

B. Khách hàng
- Đăng ký, đăng nhập
- Thêm hoặc xóa sản phẩm trong giỏ hàng 
- Đặt mua sản phẩm
- Tìm kiếm sản phẩm
- Chỉnh sửa thông tin cá nhân
